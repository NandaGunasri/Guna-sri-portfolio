Case Studies

Project 01 • Developer Platform

NeoLearn TechTrack

A real-time developer progress and learning tracking platform built to monitor, map, and analyze software engineering skill progression. It supports role profiles, concept mapping, and AI-assisted learning suggestions.

Next.js TypeScript Tailwind CSS Framer Motion Zustand Firebase Firestore Firebase Auth Google Gemini Docker GCP Cloud Run
Problem Statement
Traditional progress tracking tools remain disconnected from actual code progression and require tedious manual entry, making learning pathways static and hard to manage.
Solution
Built an interactive platform linking user progress data directly to a state store, leveraging Firebase for auth/data and Gemini for tailored guidance.
System Architecture
Developed with a Next.js framework, deploying state updates via Zustand. Persistent tracking metrics are managed in Firebase Firestore, and the backend service is containerized via Docker for GCP Cloud Run deployment.
Challenges Solved
Structured secure API communication endpoints on GCP Cloud Run, integrating Firestore event streams to update user profile pages without lag.

Project 02 • Security & Admin UI

ElectionShield SecureVote AI

A secure electronic voting management and audit application leveraging role-based controls, password hashing, and multiple database schemas to store records safely.

React.js Vite HTML5 CSS3 Node.js Express.js SQLite MySQL MongoDB JWT Bcrypt Vercel Render
Problem Statement
Administrating electronic election records requires solid access logging, multi-environment DB compatibility, and protection against unauthorized credential manipulation.
Solution
Designed a modular Express backend utilizing JWT token validation and Bcrypt hashing, connected to React client panels to manage voter roles securely.
Security Architecture
Enforces Role-Based Access Control (RBAC) via secure Express middleware. Hashed user logins are authenticated using stateless JSON Web Tokens (JWT) storage rules.
Challenges Solved
Configured a unified database abstraction layer, allowing smooth toggling between relational SQLite/MySQL engines and NoSQL MongoDB datastores during deployment.
