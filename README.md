# 🌐 Nanda Gunasri | Premium Developer Portfolio

A high-performance, recruiter-focused developer portfolio designed to resemble a modern SaaS landing page. It showcases professional engineering capabilities, flagship project case studies, and enterprise cloud/ServiceNow credentials.

🔗 **Live Website:**  
👉 [https://nandagunasri.github.io/Guna-sri-portfolio/](https://nandagunasri.github.io/Guna-sri-portfolio/)

---

## 🛠️ Architecture & Tech Stack

The portfolio is engineered with standard web semantics for peak speed, compatibility with GitHub Pages, and mobile-first responsiveness:
- **Structure**: Semantic HTML5 (optimized for SEO and screen-reader accessibility).
- **Styling**: Vanilla CSS3 (custom glassmorphic panels, CSS grid backgrounds, and a slate-dark color system with electric cyan accents).
- **Dynamic Elements**: Vanilla JavaScript (custom scroll-reveal animations, responsive mobile menu, dynamic typewriter headlines, and interactive case study deep dives).

---

## 📂 Core Sections & Features

1. **Hero**: High-impact startup landing page headline featuring dynamic role typing and quick-access call-to-actions.
2. **About & Education**: A concise summary of software development expertise, nesting academic credentials (Mohan Babu University) inside a clean, tabbed panel to preserve recruiter scanning priority.
3. **Featured Projects (Flagship Case Studies)**:
   - **NeoLearn TechTrack**: Real-time developer progress and learning telemetry platform.
   - **ElectionShield SecureVote AI**: Cryptographically secured e-voting audit database with AI-driven anomaly detection.
   - *Includes Problem Statement, Solution, Tech Badges, and interactive code/architecture deep dives.*
4. **Technical Skills**: A grouped grid of technologies (Languages, Frontend, Backend, Databases, Cloud & Platforms, Tools) displayed as clean, modern badges (no generic percentages or progress bars).
5. **Certifications**: High-visibility glassmorphism cards highlighting:
   - **AWS Certified Cloud Practitioner**
   - **ServiceNow Certified System Administrator (CSA)**
   - **ServiceNow Certified Application Developer (CAD)**
6. **Professional Experience**: A clean vertical timeline mapping key contributions from internships (AWS, ServiceNow, Front-End Dev, Agentic AI Project).
7. **Leadership & Achievements**: Impact-oriented logs covering hackathon coordination (SAMARTHA 2025) and solver recognitions.
8. **Resume**: Fast download and direct viewing links for the curriculum vitae.

## 🚀 Local Setup & Development

To test or modify the portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/nandagunasri/Guna-sri-portfolio.git
   ```
2. Navigate to the project folder and open `index.html` in your browser, or start a local dev server:
   ```bash
   npx serve
   ```
